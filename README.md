# EXThunderforest

## Description  
Unofficial repository of **Thunderforest-inspired styles**, developed to work with **Thunderforest Vector Tiles v2**.

These styles have been **tested and verified** for use in **OruxMaps (premium subscription)**, whose rendering engine is based on **Mapbox GL**.  
A **valid Thunderforest API key** is required to use these styles.

Includes:

- Styles compatible with MapLibre and Mapbox  
- Sprite generated specifically for these styles  
- Glyphs included in the repository  
- Modular design

---

## Compatibility

The Thunderforest styles are compatible with:

- **OruxMaps (premium subscription, Mapbox GL engine)**  
- **MapLibre GL JS**  
- **Mapbox GL JS**  
- Derived implementations (MapLibre Native, Mapbox Mobile, etc.)

---

## Vector Tiles  
These styles are designed to work with:

**Thunderforest Vector Tiles v2**, which include optimized layers for:

- Transportation  
- Paths and trails  
- Hydrology  
- Landcover  
- POIs  
- Relief  
- Cycleways  
- Tourist infrastructure  
- Cycling and hiking routes

> ⚠️ **Requirement:** A valid **Thunderforest API key** is needed to load these vector tiles.

---

## Sprite and Glyphs  
The repository includes:

- Sprite generated specifically for the Thunderforest styles  
- Glyphs required for text rendering  
- Guaranteed compatibility with MapLibre and Mapbox
